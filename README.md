# Credit Card Fraud Detection Dashboard

## Project Overview
This project is a Credit Card Fraud Detection Dashboard developed as part of the Codec Technologies Internship Program.

The dashboard analyzes credit card transaction data and helps identify potentially fraudulent transactions using data analytics and visualization techniques.

## Features
- Fraud transaction analysis
- Transaction monitoring dashboard
- Data visualization
- Interactive insights

## Technologies Used
- Python
- Pandas
- Streamlit
- Plotly

## Dataset
Credit Card Fraud Detection Dataset

Source:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

pip install -r requirements.txt
